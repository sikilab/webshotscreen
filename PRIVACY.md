# Privacy Policy for Full Page Capture

Effective date: 2026-04-25

Full Page Capture ("the Extension") is designed to capture and export screenshots from web pages in your browser.

## Data Collection

The Extension does not collect, store, sell, or share personal data on external servers.

All capture, stitching, compression, and export operations are processed locally in the user's browser.

## Permissions Usage

The Extension requests the following permissions only for screenshot functionality:

- `activeTab`: Access the currently active tab when the user starts a capture.
- `tabs`: Read tab and window context needed for capture flow.
- `scripting`: Inject local helper scripts to measure, scroll, and capture page content.
- `offscreen`: Process image stitching and export in an offscreen document.
- `downloads`: Save generated image files to the user's device.
- `clipboardWrite`: Copy generated image data to the system clipboard when requested by the user.
- `storage`: Save user preferences locally (for example format, resolution, and capture options).

## Data Sharing

The Extension does not transfer user data to third parties.

## Remote Code

The Extension does not execute remote code. Executable code is packaged with the extension.

## User Controls

Users control when captures start, what area is captured, export format, and whether clipboard copy is used.

Users can remove the Extension at any time from Chrome Extensions settings.
